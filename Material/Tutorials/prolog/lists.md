# Prolog

## lists

---

a **list** is a sequence of any number of terms

```prolog
[1, 2, 3]
[a, b, c, d]
[a, [1, 2, 3], tom, 15, date(1, may, 1995)]
```

<!--vert-->

a list is actually a pair of a head and a tail

```prolog
[Head | Tail]
[A, B, C] = [A | [B | [C | []]]] = [A, B | [C]]
```

---

### length/2

```prolog
length([], 0).
length([_|Tail], N) :-
    length(Tail, N1),
    N is 1 + N1.
```

```prolog
length([a, b, [c, d], e], N).
% N = 4.

length(L, 4).
% L = [_10798, _10804, _10810, _10816].
```

<!--vert-->

### length with CLP(FD)

```prolog
length([], 0).
length([_|Tail], N) :-
    N #= N1 + 1,
    length(Tail, N1).
```
